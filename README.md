# huarongdao
命令行输入

          go get github.com/nsf/termbox-go

          go get github.com/gitxiaolin/huarongdao
          
          cd $GOPATH/src/github.com/gitxiaolin/huarongdao/
          
          go run main.go
          
          
   上下左右移动光标，空格选中，然后再按上下左右移动方块。
   
   ctrl+Q 退出程序。
   
   
注：不支持winodws
